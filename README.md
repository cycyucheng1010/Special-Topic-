## 前言
在台灣45-64歲的阿茲海默症人口約為11310人，加上65歲以上的患者，於民國109年12月底共計303271人，佔全國的1.29%，換句話說每77人便有1人是阿茲海默症患者，由此可知此疾病之可怕，為此我們希望透過機器學習方法去辨別患者是否患此疾病，讓患者盡早就醫。
